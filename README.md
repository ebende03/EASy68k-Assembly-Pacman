# **How to Run the Program**

Before running this project you should ensure that the EASy68k simulator is installed on your computer, otherwise you will not be able to run this file. You can find the link to download EASy68k here: 

http://easy68k.com/

Once EASy68k is downloaded and setup, simply open the file, find the assemble source button (under project), and after it has assembled find the run button and enjoy.
